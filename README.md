# Blog Preview Card

This project is a simple blog preview card built with HTML and CSS. The assignment was to recreate a clean article card layout with an image, category label, published date, title, short description, and author details.

## Assignment Overview

The goal of this assignment was to practice building a small, polished UI component using core frontend skills. The card is centered on the page, uses a bright yellow background, includes a strong shadow effect, and responds to hover interactions.

## What I Learned

From this assignment, I learned how to structure a webpage using semantic HTML elements and how to style a card component with CSS. I practiced using flexbox to center content, adding spacing with margins and padding, and improving the design with borders, border-radius, shadows, and hover effects.

I also learned how important small details are in frontend design. Things like image sizing, font weight, colors, and spacing can make a simple component look much more professional. This assignment helped me understand how HTML provides the structure of a page while CSS controls the visual presentation.
